Seamlessly sync your WordPress media library with Dropbox. Automatically backup and organize your files in the cloud.

== Description ==

FileBird Dropbox Sync Pro extends the powerful FileBird media library organizer with secure Dropbox integration. Keep your WordPress media files synchronized with Dropbox for automated backups, easy sharing, and improved media management.

**Key Features:**

* Two-way synchronization between WordPress media library and Dropbox
* Automated scheduled backups
* Selective folder synchronization
* Preserve folder structure in Dropbox
* File version history and recovery
* Custom sync rules and filters
* Bandwidth throttling options
* Detailed sync logs and notifications
* Secure OAuth2 authentication with Dropbox

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/filebird-dropbox-sync-pro` directory, or install the plugin through the WordPress plugins screen
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to FileBird settings and connect your Dropbox account
4. Configure your sync preferences and schedule

== Frequently Asked Questions ==

= Does this plugin require FileBird? =

Yes, FileBird Dropbox Sync Pro is an extension for FileBird and requires FileBird to be installed and activated.

= Will this sync my existing media files? =

Yes, you can choose to sync your existing media files during the initial setup or run a manual sync later.

= Is there a file size limit? =

The file size limit depends on your Dropbox account type and available space.

= Can I sync only specific folders? =

Yes, you can select which FileBird folders to sync with Dropbox.
